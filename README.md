# pan-website
Creating a live webpage to track PAN

## Installation of Development Env
Run:
```
npm install
npx vite
```

Note: tested working on `npm@8.19.3` and `node v16.19.0`

## Development

### Web:
In a terminal run:
```
npx vite
```

### Server:
```
flask run
```
This should start a development build accessible on localhost.
